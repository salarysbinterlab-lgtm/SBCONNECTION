# SB Connect Stable Clean

ตัวนี้ตัด Visual Low-code Editor ออก เพื่อให้ระบบหลักกลับมารันหน้าแรกได้ก่อน

## แกนแอป

พนักงาน:
- Login
- ทำกิจกรรม
- สะสมแต้ม
- แลกของ
- อ่านข่าว
- Ranking

Admin:
- เพิ่ม/แก้พนักงาน
- เพิ่ม/แก้ข่าว
- เพิ่ม/แก้กิจกรรม
- เพิ่ม/แก้รางวัล
- ดู ledger แต้ม

## วิธีใช้

แตกทับ:

```text
D:\Projectsbconnect_app
```

รัน:

```bat
REMOVE_VISUAL_EDITOR_AND_RUN.bat
```

หรือรันสะอาด:

```bat
RUN_STABLE_CLEAN.bat
```

## Supabase config

แก้:

```text
public/app/assets/js/sbConfig.js
```
